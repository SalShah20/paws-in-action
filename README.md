# Paws in Action

**Paws in Action** is a student-led nonprofit organization dedicated to improving animal welfare through fundraising, donation drives, and community engagement across California.

## About

Paws in Action is a student-led initiative dedicated to supporting animal shelters and rescue organizations through fundraising, donation drives, and community events. Our goal is to bring students and communities together to raise resources, awareness, and support for animals in need.

### Founder

Paws in Action was founded by **Aarini Shah**, a student at Milpitas High School, who has a strong interest in animal welfare and community service. After learning about the number of animals entering shelters each year and the challenges shelters face in providing care and resources, Aarini started Paws in Action to help connect students with opportunities to support local animal welfare organizations.

### Our Vision

Our vision is to build long-term partnerships with animal shelters and rescue organizations while empowering students to make a meaningful impact through service and community engagement.

### What We Do

- **Shoe Drive Fundraisers** — Collect gently used shoes to raise funds for animal shelters and rescue organizations.
- **Pet-Themed Sock Sales** — Fun paw-print socks with all proceeds supporting animal welfare programs.
- **Blanket & Supply Drives** — Gather blankets, toys, and essentials for shelter animals awaiting their forever homes.
- **Thanksgiving Pet Food Drive** — Holiday pet food collection ensuring shelter animals and pets in need are fed.

### Our Partners

- [House Rabbit Society](https://www.rabbit.org/) — Dedicated to the rescue and welfare of domestic rabbits through education and advocacy.

## Tech Stack

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) — Build tool and dev server
- [Tailwind CSS](https://tailwindcss.com/) — Utility-first styling
- [shadcn/ui](https://ui.shadcn.com/) — Accessible UI components built on Radix UI
- [React Router](https://reactrouter.com/) — Client-side routing

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+) or [Bun](https://bun.sh/)

### Installation

```sh
# Clone the repository
git clone https://github.com/your-username/paws-in-action.git
cd paws-in-action

# Install dependencies
bun install   # or: npm install

# Start the development server
bun run dev   # or: npm run dev
```

The site will be available at `http://localhost:8080`.
